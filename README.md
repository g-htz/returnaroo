# 🦘 Returnaroo

Australia's community-powered lost & found platform.

🌏 Live Demo: https://g-htz.github.io/returnaroo/

---

## Features

- 🔍 Search lost & found listings
- 📍 Australia-wide community posts
- 📱 Fully responsive mobile-first UI
- ➕ Post lost or found items
- 💾 Local storage persistence
- 🛡️ Safety-first handover guidance
- ⚡ Built with React + Vite

---

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Lucide React
- GitHub Pages

---

## Running locally

```bash
npm install
npm run dev
```
